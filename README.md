# 🇬🇷 Учим греческий

Приложение для изучения греческих слов с использованием интервального повторения (spaced repetition).

## Особенности

- 📚 Словарь с сортировкой
- ➕ Добавление новых слов
- 🏋️ Тренировка с интервальным повторением
- 📊 Статусы слов: new → to_learn → to_check → 1_day → 2_day → 3_day → 5_day → 7_day → 12_day → 20_day → 30_day → 60_day
- 🔄 Автоматическое обновление статусов
- 🎯 Лимит 10 новых слов в день

## Технологии

- Python + Flask
- SQLite
- HTML/CSS + JavaScript
